# Code of Conduct

This project uses the Contributor Covenant Code of Conduct, version 2.1.

## Standard

Be direct, technical, and respectful. Challenge ideas and assumptions without attacking people.

## Scope

This applies to project issues, pull requests, discussions, and related public project spaces.

## Enforcement

Maintainers may remove abusive, harassing, or bad-faith participation. Serious or repeated violations may lead to a ban from project spaces.
