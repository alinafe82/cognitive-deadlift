# Paid Offer

## Cognitive Deadlift Workflow Audit

For teams using AI coding assistants who want stronger review, debugging, and reasoning workflows without adding a heavy process framework.

**Fixed fee**: USD 2,500

**Best fit**:

- A 3 to 20 person engineering team already using Codex, Claude, Gemini, ChatGPT-style assistants, or agentic coding tools.
- A team that sees faster code generation but weaker problem framing, diff review, debugging notes, or test discipline.
- A lead who wants concrete reasoning evidence in AI-assisted work, not a generic AI adoption workshop.

**What is included**:

- One 60-minute intake call about current AI-assisted engineering workflows.
- Review of up to three recent AI-assisted pull requests, tickets, or change writeups.
- Mapping from observed gaps to relevant Cognitive Deadlift skills, hooks, and review gates.
- A written audit memo with the top 5 workflow risks, recommended interventions, and adoption order.
- One 60-minute readout call with the maintainer.

**Deliverables**:

- Reasoning evidence gap analysis.
- Recommended skill workflow for the team.
- Example thinking ledger prompt or template adapted to the team's workflow.
- 30-day adoption plan for adding useful friction at review, debugging, or planning points.

**Not included**:

- Security penetration testing.
- Production incident response.
- Private codebase implementation work.
- A guarantee that an AI assistant will follow every instruction perfectly.
- Commercial relicensing of the repository content.

Commercial licensing or custom implementation can be scoped separately.

Contact the maintainer through the GitHub profile listed in `.codex-plugin/plugin.json` with the subject `Cognitive Deadlift Workflow Audit`.
