# Licensing

Cognitive Deadlift uses a split license to keep the project usable while protecting the written methodology.

## Code License

Executable and operational files are licensed under MIT:

- Scripts
- Hooks
- GitHub Actions workflows
- Plugin manifests
- Build and validation configuration

This keeps installation, validation, and integration simple.

## Content License

Skills, docs, prompts, catalogs, and written methodology are licensed under CC BY-NC-SA 4.0.

This means people can study, share, and adapt the content with attribution, but they cannot commercially repackage it or close off derivatives without separate permission.

## Commercial Use

Commercial use of the CC BY-NC-SA content requires separate permission from the maintainer.

The MIT-licensed code can be used commercially under the MIT terms.

## Contributor Rule

Contributions are accepted under the same split-license model. If you contribute a skill or documentation, it is CC BY-NC-SA 4.0. If you contribute executable tooling, it is MIT.
